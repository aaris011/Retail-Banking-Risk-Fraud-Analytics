# Business Questions

## Retail Banking Risk & Fraud Analytics Platform

---

# Objective

The objective of this project is to help banks identify financial risks, detect fraudulent activities, and improve customer decision-making using data analytics and machine learning.

---

# Fraud Detection Questions

### Transaction Analysis

* What percentage of all transactions are fraudulent?
* Which transaction amounts are most likely to be fraudulent?
* Which product categories experience the highest fraud rate?
* Which payment methods are more vulnerable to fraud?
* Do fraudulent transactions occur more frequently at certain times?

---

### Customer Analysis

* Which customer segments have the highest fraud probability?
* Does device type influence fraud occurrence?
* Does customer location affect fraud risk?
* Which card types are targeted most frequently?

---

### Risk Analysis

* Which features contribute most to fraud detection?
* Can fraud be predicted before transaction completion?
* Which variables have the highest predictive power?

---

# Loan Default Risk Questions

### Customer Profile

* Which customers are most likely to default on loans?
* How does customer income affect default risk?
* Does employment history influence repayment behaviour?
* How does age impact loan repayment?

---

### Financial Behaviour

* Do previous loans increase default probability?
* How does credit history affect loan approval?
* Which customers have multiple outstanding loans?
* Does previous repayment behaviour predict future defaults?

---

# Business Intelligence Questions

* What is the overall financial risk level of customers?
* Which customer segments should receive closer monitoring?
* What factors should be considered before approving a loan?
* How can fraud losses be minimized?
* Which KPIs should be displayed in management dashboards?

---

# Expected Outcomes

The project should enable banks to:

* Detect fraudulent transactions more accurately.
* Predict loan defaults before approval.
* Reduce financial losses.
* Improve customer risk assessment.
* Support data-driven decision-making.
* Build interactive dashboards for business users.
